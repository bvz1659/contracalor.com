Features

Everything a viticulture professional needs.

Built by a Napa Valley wine professional, for the people who live and breathe the growing season.

📍
Pin Any Vineyard
Drop a pin anywhere in the world. Station-priority weather data pulls from NOAA, ASOS, and Weather Underground networks.
🌡️
Winkler Classification
Automatic Region I–V classification using the standard base 50°F formula. Apr 1 – Oct 31, consistent with CIMIS and UC Davis reporting.
📊
Vintage Comparison
Compare up to 3 vintages side by side with accumulation curves aligned from Day 1. Instantly see how this year stacks up.
🍇
Vineyard Profiles
Save multiple vineyard locations with nested vintage history. Load any saved vineyard directly into the tracker in one tap.
🔄
Live Season Tracking
Current season updates automatically each day using delta fetching — only new days are pulled, keeping API usage minimal.
🌏
Both Hemispheres
Full Southern Hemisphere support with harvest year conventions. Oct 1 – Apr 30 season with automatic hemisphere detection.

Contra Calor uses Visual Crossing's station-priority weather API — pulling from real physical observation stations closest to your vineyard, not interpolated grid models.

Validated against CIMIS (California Irrigation Management Information System) stations at Oakville and Bennett Valley, and against a Weather Underground personal weather station near Placerville in the Sierra Foothills.

Average variance across validated sites: 3.8% — well within acceptable range for a viticulture decision-support tool.
